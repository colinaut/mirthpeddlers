Fist of Liberty

Posted by Colin Fahrion on
 January 28, 2018

UPDATE Oct 15, 2019: This game is on hold indefinitely. While I had some fun play test sessions, I realized the reason the games were fun more had to do with my ability to run a good session and not because the rules supported fun play. There is still some good in there which I may repurpose or release in some form at some point.

Introducing a new game I’m working on called Fist of Liberty!

Fist of Liberty is a tabletop roleplaying game about normal people attempting to overthrow the oppressive state. Inspired by such dystopian classics such as 1984 and Brave New World, and films like eTHX-1138, Brazil, Max Headroom, Idiocracy, Equilibrium, and The Hunger Games. It is a one shot game designed to be played in 4-6 hours. Every game of Fist of Liberty is different as it starts with a fast cooperative world building and character building session!

The game mechanics for Fist of Liberty are inspired by the excellent game Soth by Steve Hickey and also by Apocalypse World. If you haven’t played either I highly recommend both!

I’ve already started play testing the game. Several of the play tests happened at Big Bad Con 2017 which were a whole lot of fun and very instrumental in helping me improve the game and develop it further. I will be running additional play test sessions sometime soon so if you are interested in joining please drop me a line at info@mirthpeddlers.com
